# Dotfiles.

Dotfiles for:
 * Vim
 * Mutt
 * Zsh


## Installing

Clone repo with submodules:

```
cd ~
git clone --recursive https://github.com/w1zdun/dotfiles.git
```

Move dotfiles to home:
```
cd dotfiles
mv .* ~
rm -fr dotfiles
```
